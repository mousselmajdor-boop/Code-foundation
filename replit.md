# Workspace

## Overview

Python Streamlit starter template.

## Stack

- **Language**: Python 3.11
- **Framework**: Streamlit
- **Entry point**: `app.py`
- **Config**: `.streamlit/config.toml`

## Run

The "Start application" workflow runs:

```
streamlit run app.py --server.port 5000
```

The app listens on port 5000 and is served through the Replit preview pane.
